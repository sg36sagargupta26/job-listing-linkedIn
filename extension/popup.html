📋 How to set up Google Sheets
  1. Create a Google Sheet with headers:
    Title | Company | Location | URL | Posted | Description | Match% | Matched Skills | Missing Skills | Summary | Saved At
  2. Go to Extensions → Apps Script
  3. Replace the code with the apps-script.gs file from this project
  4. Click Deploy → New deployment → Web app
  5. Set "Who has access" = Anyone
  6. Copy the Web App URL and paste above
